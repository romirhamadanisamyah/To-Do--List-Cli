# To-Do List App CLI ✅

Aplikasi terminal untuk manajemen tugas harian. Project ke-2 untuk mengasah logika CRUD dengan Python.

## ✨ Fitur
- **Create**: Tambah tugas baru
- **Read**: Lihat semua daftar tugas
- **Update**: Tandai tugas yang sudah selesai [ ]
- **Delete**: Hapus tugas yang tidak perlu
- **Penyimpanan**: Data otomatis tersimpan di `tugas.txt`

## 🛠️ Teknologi & Konsep
- Python 3
- List of Dictionary
- CRUD Operation: Create, Read, Update, Delete
- File Handling `.txt`

## ▶️ Cara Menjalankan
1. `python main.py`
